Detailed Description

The Staff class is a derived class of Members. It's used to represent a staff member of a gym. On top of the datas used in a Member type, a staff must specify his/her employee number (unsigned long employee_id), as well as its role in the gym (Clearance staff_clearance).

The role is represented using a enum Clearance, and has the values of GENERAL, MANAGER, ADMINISTRATOR.

The Staff class requires to be manually initialized after construction, since the constructor does not take any arguments. Use the "intialize" function to fill in the data for the Staff. Alternatively, each of the data fields can be set individually using individual set methods.
