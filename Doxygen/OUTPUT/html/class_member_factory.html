Detailed Description

The MemberFactory class makes clones of a templated Customer and Staff to return new objects back to the caller. Two templates are created, Customer* CustomerClone and Staff* StaffClone. They are both initialized upon constructing the factory.

Once an instance of the MemberFactory has been constructed, copies of Customer can be created by calling the getCustomer() function. Similarily, copies of Staff can also be created by calling the getStaff() function. Both functions will only return a barebone template. The corresponding "initialize" function for each class must be called after to fill them with data.
